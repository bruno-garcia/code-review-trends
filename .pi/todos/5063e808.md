## Task
Update the app's ClickHouse query layer to support products. Add Product type, product-level aggregation queries. Keep bot-level queries for detail views.

## Files
- `app/src/lib/clickhouse.ts` (modify)

## Details

### New types:
```ts
export type Product = {
  id: string;
  name: string;
  website: string;
  description: string;
  brand_color: string;
  avatar_url: string;
};
```

### New/updated queries:
1. `getProducts()` — fetch all products
2. `getProductById(id)` — single product
3. `getProductSummaries()` — like getBotSummaries but aggregated by product_id (JOIN bots b → products p, GROUP BY p.id)
4. `getWeeklyActivityByProduct(productId?)` — weekly activity summed across all bots in a product
5. `getProductComparisons()` — like getBotComparisons but product-level
6. `getProductBots(productId)` — list bots for a product with their individual stats (for detail page breakdown)

### Keep existing bot-level queries:
They're still useful for the product detail page where you show per-bot breakdown.

### Key pattern:
```sql
SELECT p.id, p.name, SUM(ra.review_count) ...
FROM review_activity ra
JOIN bots b ON ra.bot_id = b.id
JOIN products p ON b.product_id = p.id
GROUP BY p.id, p.name
```
