# Code Review Trends

Tracking the adoption of AI code review bots on GitHub.

**https://codereviewtrends.com**

## Quick Start

```bash
npm install
npm run dev        # Starts ClickHouse + Next.js dev server
```

Open http://localhost:3000.

## Running Tests

```bash
npm run test:e2e   # Playwright e2e tests (needs ClickHouse running)
```

See [AGENTS.md](AGENTS.md) for architecture details and contributing guidelines.
